# Theory-guided meta-analysis plan

## Primary comparative question

Why are some flowering-plant species polymorphic in floral colour whereas others are monomorphic?

The meta-analysis is designed to test predictions derived from the phase theory rather than to assemble an unconstrained list of correlates.

## Primary response

Species-level state:

- `1`: documented intraspecific floral colour polymorphism;
- `0`: documented monomorphism under an explicit search or sufficiently broad description;
- `NA`: insufficient evidence.

A strict sensitivity analysis should distinguish:

- population-level polymorphism;
- species-level polymorphism occurring among populations;
- continuous colour variation;
- discrete colour morphs.

These states should not be collapsed blindly.

## Core theory-to-data mapping

The two-patch result is

\[
 b + H_{\mathrm{eff}}(h,m) > |d|,
\qquad
H_{\mathrm{eff}}=\sqrt{h^2+m^2}-m.
\]

The empirical model should therefore prioritize the following quantities.

### Balancing-selection proxies (`b_proxy`)

Examples:

- direct evidence of negative frequency-dependent selection;
- rare-morph reproductive advantage;
- pollinator switching or avoidance that generates rare-morph advantage;
- antagonistic agents producing reciprocal fitness advantages.

### Spatial heterogeneity proxies (`h_proxy`)

Examples measured across the species range or sampled populations:

- climatic variance;
- elevational breadth or variance;
- habitat diversity;
- pollinator-community beta diversity;
- spatial variation in morph-specific selection where directly measured.

### Homogenization / connectivity proxies (`m_proxy`)

Examples:

- direct estimates of gene flow;
- population genetic connectivity;
- range continuity versus fragmentation;
- dispersal distance proxies;
- island/mainland or highly isolated population structure.

### Directional-selection proxies (`d_proxy`)

Examples:

- consistent reproductive advantage of one morph across sites;
- consistent survival advantage;
- abiotic stress strongly favoring one pigment phenotype;
- repeated directional transitions toward one colour state.

## Primary hypothesis

At minimum, fit a phylogenetically informed binary-response model:

\[
\operatorname{logit} P(Y_i=1)
=
\alpha
+\beta_h H_i
+\beta_m M_i
+\beta_{hm}H_iM_i
+\beta_b B_i
+\beta_d D_i
+u_{\mathrm{phylogeny}}.
\]

Theory predicts:

- `beta_h > 0`;
- `beta_hm < 0` when larger `M` means stronger connectivity/gene flow;
- `beta_b > 0`;
- `beta_d < 0`.

The heterogeneity-by-connectivity interaction is the sharpest prediction generated by the current spatial model.

## Preferred analyses

1. Phylogenetic logistic regression or Bayesian phylogenetic generalized linear mixed model.
2. Genus/family random effects when a complete phylogeny is unavailable.
3. Observation-process sensitivity analysis for uneven study effort.
4. Matched comparisons within genera where polymorphic and monomorphic species coexist.
5. Separate analyses for discrete polymorphism and continuous variation.

## Detection and ascertainment bias

A species coded as monomorphic may simply be poorly sampled. Record:

- number of populations examined;
- geographic coverage;
- sample size where available;
- source type: targeted polymorphism study, flora, monograph, taxonomic treatment, occurrence database, image survey;
- whether absence of variation was explicitly stated.

Use study effort as a covariate or model a latent true state with imperfect detection if data quality permits.

## Evidence hierarchy

Use three nested datasets.

### Tier 1 — strict

Species with explicit population sampling and a defensible polymorphic or monomorphic classification.

### Tier 2 — comparative

Add floras, monographs, taxonomic treatments, and broad surveys with clear colour descriptions.

### Tier 3 — exploratory

Add occurrence-image and citizen-science derived evidence, but model observation effort explicitly.

The main paper should not let the largest but noisiest dataset determine the result alone.

## Staged workflow

### Stage A — theory recovery

Complete the analytical extensions needed to identify the minimal comparative predictions.

### Stage B — scoping dataset

Assemble a pilot dataset across a deliberately broad taxonomic sample to test coding feasibility and effect directions.

### Stage C — preregistered systematic review / meta-analysis

Freeze search strings, inclusion criteria, coding rules, phylogenetic strategy, primary model, and sensitivity analyses.

### Stage D — pattern test

Test whether polymorphism prevalence follows the predicted phase structure, especially the heterogeneity-by-connectivity interaction.

## Current falsifiable prediction

The strongest immediate prediction is:

> Environmental heterogeneity should be associated with a higher probability of floral colour polymorphism, but this association should weaken as connectivity or gene flow increases.

A null or reversed interaction would directly challenge the current two-patch theory and motivate model revision rather than post hoc mechanism shopping.
