# Empirical data progress

## Current state

The empirical program now has three distinct layers:

1. **Verified evidence ledger** — species promoted only when the outcome class is supported strongly enough for analysis.
2. **High-priority screening audit** — candidate species actively checked for whether evidence supports local coexistence, geographic mosaic, mixed structure, or only species-level colour variation.
3. **Full angiosperm census workflow** — every accepted species begins as `unknown`; no absent hit is converted to monomorphism.

The current verified ledger contains:

- `Gymnadenia rhellicani` — local coexistence
- `Silene littorea` — local coexistence
- `Linanthus parryae` — mixed spatial polymorphism

These remain the only rows promoted to `verified` in the evidence ledger at present.

## First active screening wave

The first high-priority audit has now been completed for 10 additional candidate systems.

A key result of this audit is that **species-level colour variation is much easier to establish than the spatial outcome class required by the theory**. Several species clearly show more than one flower colour, but available evidence inspected so far does not yet distinguish:

- local coexistence,
- geographic mosaic,
- mixed structure, or
- horticultural / environmentally induced variation.

Therefore these cases remain outside the verified analytical dataset until population-level evidence is extracted.

The strongest current promotion candidate is `Raphanus sativus`, for which a primary study explicitly examines petal-colour variants in wild populations and pollinator responses to colour morphs. Full-text population-structure coding is still required before final promotion.

## Operational rule

A species is not promoted merely because multiple colours are described. Promotion requires evidence matching the target outcome:

- `local_coexistence`: multiple inherited colour morphs documented within the same natural population;
- `geographic_mosaic`: different populations dominated by different morphs, with little or no local coexistence;
- `mixed`: both differentiated populations and locally polymorphic populations documented within the species;
- `monomorphic_confirmed`: explicit broad evidence supporting absence of discrete flower-colour polymorphism, with sufficient observation effort.

`unknown` is never treated as `monomorphic_confirmed`.
